# Private Context Workflow

This repository is public-facing. Private assessment material must stay outside the repository.

## Recommended Google Drive Setup

Create a restricted Google Drive folder named:

```text
MTD367 Private Assessment Context
```

Use it for private course-planning and assessment-alignment material, such as:

- TMA and ECA question drafts
- marking guides
- rubrics
- model answers
- internal assessment alignment notes

Do not share this folder publicly. Do not use "anyone with the link." Share it only with authorized course staff.

## Using Private Docs With Codex

When a task needs assessment alignment, refer to the private folder or document by name in the prompt. For example:

```text
Use the restricted Google Drive folder "MTD367 Private Assessment Context" to align the 2026 lesson plan with the assessments. Do not copy assessment questions, rubrics, marking guides, or model answers into the public repo.
```

Codex may use private docs to derive:

- public learning outcomes
- topic coverage
- review themes
- common mistakes to address
- practice-style examples that do not reveal assessment content

Codex must not reproduce private assessment text in public files.

## Public Output Rules

Before adding or updating public materials, check that the output:

- does not include TMA/ECA questions
- does not include marking guides, rubrics, or model answers
- does not include screenshots or exports from private assessment docs
- does not include private Google Drive or Canva links
- is suitable for students to read before assessment submission

Post-submission review notes may be public when they are written as learning resources and intentionally approved for release.
