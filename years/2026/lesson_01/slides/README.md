# Lesson 01 Slides

Canva is the source of truth for these slides.

## Focus

Introduce the course, the development environment, and the minimum Swift syntax students need before building app interfaces.

## Learning Outcomes

- Explain the course workflow, lesson structure, and public material locations.
- Set up and navigate Xcode for simple Swift exercises.
- Use variables, constants, basic types, functions, conditionals, and loops.
- Read and run small Swift examples with clear naming and formatting.

## Rough Deck Flow

1. Course welcome and expectations.
2. How the 2026 lessons build toward app development.
3. Xcode orientation and running a first Swift example.
4. Swift values, types, and simple expressions.
5. Functions, decisions, repetition, and small problem-solving patterns.
6. Hands-on practice with short runnable exercises.
7. Summary and preview of data modelling in Lesson 02.

## Demo And Practice Direction

Use compact Swift Playground or Xcode examples that students can modify quickly. Practice should emphasize reading code, predicting output, and making small changes rather than building a full app yet.

## Assessment Preparation

This lesson builds core Swift confidence for later public practice and TMA01 readiness. Do not include private assessment questions or model answers.
