# AGENTS.md

## Repository Purpose

- This is a public-facing teaching repository for MTD 367 iOS Application Development.
- Keep public lesson materials, slide exports, review notes, and Swift/iOS example code here.
- Do not add private course administration or private assessment material to this repository.

## Public Assessment Safety

- Never add TMA/ECA questions, marking guides, rubrics, model answers, or private assessment files.
- Private Google Docs/Drive assessment material may be used only as context for designing public learning materials.
- Before using private Drive docs, confirm the task requires assessment alignment.
- Derive learning outcomes, topic coverage, review themes, and practice-style examples from private context, but do not reproduce private assessment text.
- Before finalizing changes, check that no private assessment text or files were added to the public repo.

## Organization

- Organize materials by year, then lesson.
- Keep archived 2025 materials under `years/2025/`.
- Keep 2026 work under `years/2026/`.
- Keep code examples under the matching lesson's `code/` folder.
- Keep post-submission review notes under the matching year's `reviews/` folder.

## Slides

- For 2026 onward, Canva is the source of truth for slides.
- Store Canva view/edit references and slide notes in each lesson's `slides/README.md`.
- Store student-facing exported slides in each lesson's `slides/exports/` folder.
- Do not treat Marp Markdown as the default 2026 slide source.
- Do not publish private Canva links, assessment-only slides, or private assessment screenshots.

## Code And Local Files

- Avoid committing local Xcode user files, build output, or machine-specific workspace state.
- Keep intentional Swift Playgrounds, Xcode project files, assets, and media files with the relevant lesson.
- Prefer small, lesson-focused examples over broad refactors.

## Verification

- Use `rg --files` to inspect the tree after reorganizing materials.
- Check that public docs and generated outputs do not contain private assessment content before finishing.
